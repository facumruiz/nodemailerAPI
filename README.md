### This is a NodeMailer Tutorial for beginners

### Technologies Used
- Express: ^4.17.1
- Nodemailer: ^6.4.11
- multiparty: ^4.2.2
- dotenv: ^8.2.0
- Node: 12.14.1 

### Usage
1. Install dependencies: `npm install`
2. Add an `.env` file in the root directory with the following variables:
    - `EMAIL=YOUR_EMAIL`
    - `PASS=YOUR_PASSWORD`
3. Run server: `npm start`
